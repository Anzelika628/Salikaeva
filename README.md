# Weather Diary

## Автор
Иванов Иван

## Описание программы
Приложение для ведения дневника погоды с возможностью добавления, просмотра, фильтрации записей и сохранения данных в JSON.

## Как запустить
1. Установите Python 3.x.
2. Установите зависимости:
   pip install -r requirements.txt
3. Запустите:
   python weather_diary.py

## Примеры использования
1. Добавьте запись через форму.
2. Используйте фильтры по дате и температуре.
3. Сохраните данные в файл и загрузите их обратно.
